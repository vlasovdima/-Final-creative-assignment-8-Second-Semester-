# semester-load-merge

Объединяет нагрузку двух семестров в один Excel-файл.

Работает с любыми `.xls` файлами нагрузки у которых 11 колонок (A-K):
Дисциплина, Вид нагрузки, Группы, Часы, Поток, Часы по УП, Табельный №, ФИО, Должность, Аудитория, Описание.

## Быстрый старт

```bash
# Вариант 1: файлы по умолчанию в текущей папке
python itz_consolidated.py

# Вариант 2: указать входные файлы
python itz_consolidated.py осень.xls весна.xls

# Вариант 3: указать всё
python itz_consolidated.py осень.xls весна.xls результат.xlsx
```

## Что делает

Берёт два `.xls` файла (осень + весна), объединяет их и создаёт один `.xlsx` с тремя листами:

**сводная** — все строки нагрузки из обоих файлов + колонка «семестр»

**преподаватели** — сводка по каждому: часы за осень, весну, итого, отношение к ставке (норма 830ч)

**статистика** — общее количество строк, часов, разбивка по типам нагрузки

## Как работает код

3 функции + `main()`:

**1) `read_semester()`** — читает .xls, возвращает список словарок. Пустые и итоговые строки пропускает.

**2) `calc_teacher_load()`** — суммирует часы по каждому ФИО отдельно за осень, весну и итого.

**3) `write_output()`** — создаёт .xlsx с тремя листами, форматирует заголовки, ширину колонок.

**4) `main()`** — парсит аргументы командной строки, запускает всё по порядку:
```
читаем осень → читаем весну → объединяем → считаем часы → пишем .xlsx → выводим сводку
```
